# Browser Extensions Manager UI - Frontend Mentor

This is a solution to the Browser Extensions Manager UI challenge on Frontend Mentor. It is a responsive and interactive dashboard designed to manage browser extensions, featuring dynamic filtering and theme switching.

## Features

* **Interactive Filtering:** Filter extensions by 'All', 'Active', or 'Inactive' states using JavaScript logic.
* **Dark Mode Toggle:** Users can switch between Light and Dark themes.
* **Dynamic Removal:** Remove extension cards from the DOM instantly by clicking the 'Remove' button.
* **Responsive Grid Layout:** Uses CSS Grid to adjust layout for all screen sizes.
* **Custom Toggle Switches:** Styled checkboxes that look like modern On/Off switches.

## Technologies Used

* HTML5
* CSS3 (CSS Grid, Flexbox & Variables)
* JavaScript (DOM Manipulation)
* Git & GitHub
